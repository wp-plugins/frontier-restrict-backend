== Description ==
 
Frontier Restrict Backend will restrict users from access to the backend (admin area) 

= Main Features =
* Users without the capability manage_options will not have access to the backend (admin area, and will be redirected to the home url.
 
Related plugins: [Frontier Post](http://wordpress.org/plugins/frontier-post/) , [Frontier Buttons](http://wordpress.org/plugins/frontier-buttons/) 
 
== Installation ==

1. ftp the plugin to the `/wp-content/plugins/`  directory or install it from plugin install on your site
2. Activate the plugin through the 'Plugins' menu in WordPress
